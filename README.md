# TempoTerm 官方網站

這是 [TempoTerm](https://github.com/mukiwu/tempo-term) 的官方網站原始碼

TempoTerm 是一款為開發而生的超輕量終端機軟體，把終端機、編輯器、檔案總管、Git 與 AI 助手整合進同一個視窗

## 連結

- 產品原始碼：<https://github.com/mukiwu/tempo-term>
- 線上網站：<https://mukiwu.github.io/tempo-term-website/>

## 本機預覽

單一 HTML 檔，不需要 build，用任何靜態伺服器打開即可

```bash
python3 -m http.server 8823
```

接著開啟 <http://localhost:8823>
